# Dev Dashboard — CLAUDE.md

## What This Is

ターミナルで動く開発状況ダッシュボード。Git、CI/CD、タスク管理、コード品質を一画面で把握する。

## Development Team

このプロジェクトは6人のサブエージェント開発チームで進行する。各メンバーは自律的に判断し、チーム内でコミュニケーションして開発を進める。ユーザーへの確認はプロジェクトの方向性を変える判断のみ。

チーム構成とSOUL定義は `.claude/team/` 配下を参照。

## Team Communication Protocol

- 対立が起きたらKai（PM）が30秒で裁定
- 技術判断はSora、UI判断はRin、品質判断はRyuが権限を持つ
- ユーザーエスカレーションはKaiのみが判断する
